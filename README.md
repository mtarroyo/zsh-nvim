# zsh-nvim

[oh-my-zsh plugin](https://github.com/robbyrussell/oh-my-zsh) for those who are migrating from [vim](https://github.com/vim/vim) to [nvim](https://github.com/neovim/neovim).

## Prerequisites

You should have `nvim` installed, this plugin will do nothing otherwise.

## Install

Create a new directory in `$ZSH_CUSTOM/plugins` called `zsh-nvim` and clone this repo into that directory.
```
git clone https://github.com/mtarroyo/zsh-nvim.git $ZSH_CUSTOM/plugins/zsh-nvim
```

Add `zsh-nvim` to your plugin list in `~/.zshrc`.

## Usage

This plugin will replace `vim` with `nvim`. To run the real `vim` command, you can use `rvim`.
